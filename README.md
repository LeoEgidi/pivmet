# pivmet

```pivmet``` package collects some pivotal methods to:

- undo label switching arising in Bayesian mixture models through *pivotal relabelling* (Egidi et al. 2018a)

- *pivotal seeding* for the Kmeans algorithm (Egidi et al. 2018b). 

See the vignettes in the vignettes folder for exploring how the package works!

PAY ATTENTION! BEFORE INSTALLING: make sure to download the JAGS program at
https://sourceforge.net/projects/mcmc-jags/.

If you not install it, you can not install pivmet from GitHub.
